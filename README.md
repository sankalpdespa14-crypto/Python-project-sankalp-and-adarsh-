# 🧬 DNA to Protein Translator

A simple bioinformatics tool built in Python that converts DNA sequences into proteins.

## 🚀 Features
- DNA validation
- Transcription (DNA → mRNA)
- Reverse complement
- Protein translation using codon table
- Detects multiple proteins (ORFs)

## 🧪 How to Run

```bash
python dna_translator.py
